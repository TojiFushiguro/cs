Cyber Security OverviewSafeguarding systems, networks, and data from cyber-attacks using technologies and controls. 1. Critical Infrastructure: Protects SCADA systems per NIS Regulations.
2. Network Security: Secures networks, servers, and related components.
3. Cloud Security: Safeguards data, applications, and Cloud infrastructure.
4. IoT Security: Ensures safety of IoT devices and networks.
5. Application Security: Addresses vulnerabilities in software development.

Importance1. Costs of Breaches: Lead to fines and reputational damage.
2. Sophisticated Attacks: Evolving tactics, including social engineering and malware.
3. Board-Level Focus: New regulations demand heightened oversight.
4. Cyber-Crime Impact: Global annual loss exceeds $1 trillion.
Types of cyber threatsMalware Attack: Disrupts or damages computers using malicious software.
Types of Malware:
1. Virus: Self-replicating program infecting files.
2. Trojan: Disguised as legitimate software, causing damage or collecting data.
3. Worm: Self-replicating malware spreading to other computers.
4. Spyware: Secretly records user activities, capturing sensitive information.
5. Ransomware: Locks user's files, demanding a ransom.
6. Adware: Advertising software spreading malware.
7. Botnets: Networks of infected computers used by cybercriminals.
1. SQL Injection: Exploits database vulnerabilities, stealing data by injecting malicious code.
2. Phishing Attack: Deceptive emails trick victims into providing sensitive information, often targeting credit card data.
3. Man-in-the-Middle: Intercepts communication between individuals on unsecure networks, stealing data flow.
4. Denial-of-Service: Overwhelms networks and servers with traffic, rendering systems unusable.
Cyber Security Challenges1. Ransomware: Demands ransom for data access.
2. IoT Risks: Growing security challenges.
3. Cloud Concerns: Increased data theft worries.
4. Phishing: Social engineering steals data.
5. Blockchain Threats: Compromise data and operations.
6. Software Vulnerabilities: Risks from unpatched software.
7. ML and AI Exploitation: Tech used for cyber threats.
8. BYOD Challenges: Personal device use poses gaps.
9. Insider Threats: Internal risks from data leaks.
10. Outdated Hardware Risks: Exposes to cyberattacks due to compatibility issues.
Forensics: Electronic data as crime evidence, aiming to trace & prosecute culprits.Types1. Network Forensics: Monitors and analyzes network traffic.
2. Email Forensics: Examines criminal emails and recovers threads.
3. Malware Forensics: Investigates hacking crimes by analyzing malware.
4. Memory Forensics: Collects data from memory for retrieval.
5. Mobile Phone Forensics: Analyzes data from mobile phones.
6. Database Forensics: Examines and analyzes data from databases.
7. Disk Forensics: Extracts data from storage media.
Importance• Collects digital evidence for criminal tracing.

• Used in solving digital and real-world crimes.

Capabilities:
